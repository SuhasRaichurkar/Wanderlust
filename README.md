# WanderLust

**WanderLust** is a full-stack travel web application.  
It allows travelers to explore destinations, book hotels, and connect with hosts, providing a seamless and interactive user experience.

---

## Live Demo

[ View WanderLust Live](https://wanderlust-2-mf26.onrender.com)

---

## Features

- User authentication with signup/login  
- Create, read, update, and delete listings (CRUD operations)  
- Like/save listings using a heart icon  
- View and submit reviews for destinations and hotels  
- Interactive map integration using Mapbox with markers and popups  
- Responsive design with Bootstrap and custom CSS  
- Smooth animations using GSAP and ScrollTrigger  
- Sections include Hero, Categories, Services, Testimonials, Traveler vs Host comparison  
- Optional skeleton loaders and animated components for better UX  

---

##  Tech Stack

- **Frontend:** HTML, CSS, Bootstrap, JavaScript, EJS (template engine)  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Cloudinary:** For image storage and upload  
- **Authentication & Security:** JWT  
- **APIs & Tools:** Mapbox, GSAP, ScrollTrigger  
- **Version Control:** Git & GitHub

##  Installation

1) Clone the repo:  
```bash
git clone https://github.com/pradnyajadhav15/WanderLust.git
cd WanderLust Project

2) Install dependencies: npm install

3) Create a .env file in the root directory and add:

MONGODB_URI=<your_mongodb_connection_string>
JWT_SECRET=<your_jwt_secret>
CLOUDINARY_CLOUD_NAME=<your_cloud_name>
CLOUDINARY_API_KEY=<your_api_key>
CLOUDINARY_API_SECRET=<your_api_secret>

4) Run the project: node index.js

5) Open in browser: http://localhost:8080
